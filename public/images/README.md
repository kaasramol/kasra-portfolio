# UI/UX Image Assets

Drop your Figma exports here. Each folder corresponds to a case study.

## Folder Structure

### /onboard
- `hero.png` — Main project hero image (1600x900)
- `before-after.png` — Before/after comparison
- `journey-map.png` — Current-state journey map
- `wireframes.png` — Key wireframe screens
- `hifi-1.png` to `hifi-4.png` — Hi-fi screen designs
- `prototype.png` — Prototype screenshot or GIF
- `results.png` — Usability test results chart

### /teampulse
- `hero.png` — Main project hero image (1600x900)
- `competitive.png` — Competitive analysis matrix
- `ia.png` — Information architecture diagram
- `dashboard-hr.png` — HR admin dashboard
- `dashboard-manager.png` — Manager team view
- `survey-mobile.png` — Employee survey on mobile
- `insight-card.png` — The Insight Card component
- `design-system.png` — Design system overview

### /focusflow
- `hero.png` — Main project hero image (1600x900)
- `journey-map.png` — Day-in-the-life journey map
- `mobile-home.png` — Mobile home screen
- `mobile-session.png` — Active focus session (mobile)
- `mobile-complete.png` — Session complete celebration
- `web-dashboard.png` — Web dashboard
- `web-analytics.png` — Analytics view
- `design-system.png` — Design system overview

### /about
- `headshot.jpg` — Your profile photo (800x800, square)

## Export Tips
- Export at 2x from Figma for retina displays
- Use PNG for UI mockups, JPG for photos
- Keep hero images around 1600x900px
- Compress with TinyPNG before adding
